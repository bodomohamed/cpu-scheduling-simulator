 # 🚀 CPU Scheduling Simulator 

A modern interactive web application to simulate and compare CPU scheduling algorithms:

- 🔵 Round Robin (RR)
- 🟢 Shortest Job First (SJF)

---

## 🎯 Features

- Dynamic process input
- Time Quantum validation
- Real-time scheduling simulation
- Timeline-based Gantt Charts
- Performance metrics:
  - Waiting Time (WT)
  - Turnaround Time (TAT)
  - Response Time (RT)
- 📊 Line Chart comparison
- ⚖️ Automatic algorithm analysis
- 📄 Export results as PDF
- 🧪 Predefined test scenarios

---

## 📊 Algorithms Overview

### 🔵 Round Robin
- Fair scheduling
- Uses time quantum
- Good response time

### 🟢 Shortest Job First (SJF)
- Minimizes waiting time
- Efficient for short jobs
- May cause starvation

---

## 🧪 Test Scenarios

- Scenario A: Mixed workload
- Scenario B: Short jobs
- Scenario D: Long job sensitivity

---

## 📈 Comparison Insights

| Metric | Better Algorithm |
|------|----------------|
| Waiting Time | SJF |
| Turnaround Time | SJF |
| Response Time | Round Robin |
| Fairness | Round Robin |
